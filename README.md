# Speed Test ke UIII.ac.id

Aplikasi web untuk mengukur kecepatan internet ke domain **uiii.ac.id**. 

## Fitur

✅ **Test Latency/Ping** - Mengukur waktu respons ke uiii.ac.id  
✅ **Test Download Speed** - Mengukur kecepatan download  
✅ **Test Upload Speed** - Simulasi kecepatan upload  
✅ **Visualisasi Grafik** - Chart real-time dengan Chart.js  
✅ **History Test** - Menyimpan 20 test terakhir di localStorage  
✅ **Export Data** - Export ke CSV dan JSON  
✅ **Dark Mode** - Toggle mode gelap/terang  
✅ **Auto Monitoring** - Test otomatis dengan interval tertentu  
✅ **Responsive Design** - Mobile-friendly

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Klik tombol **"Start Speed Test"**
3. Tunggu proses testing selesai
4. Lihat hasil di grafik dan tabel history
5. Export data jika diperlukan

## Teknologi

- HTML5
- CSS3 (dengan custom styling)
- JavaScript ES6+
- Chart.js untuk visualisasi
- LocalStorage untuk menyimpan data

## Catatan

- Aplikasi ini mengukur koneksi ke domain **uiii.ac.id**
- Hasil dapat bervariasi tergantung kondisi jaringan
- History disimpan di browser (maksimal 20 test)
- Data history dapat dihapus atau di-export
